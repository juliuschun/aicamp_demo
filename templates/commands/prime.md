이 프로젝트를 시작하기 전에 맥락을 파악할게:

1. **CLAUDE.md** 파일 읽기 (있으면)
2. **codify.md** 파일 읽기 (있으면)
3. 최근 변경된 파일 5개 확인
4. 폴더 구조 파악

파악한 내용을 간단히 요약해줘:

```
📁 프로젝트: [이름]
📝 설명: [한 줄 설명]
🔄 최근 작업: [최근에 뭘 했는지]
⚠️ 주의 사항: [codify에서 발견한 주의점]
📂 구조: [주요 폴더 설명]
```
