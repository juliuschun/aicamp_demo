---
description: 방금 한 작업에서 배운 것을 codify.md에 기록
argument-hint: [배운 내용 또는 "자동"]
---
방금 한 작업에서 배운 것을 기록할게:

1. 최근 대화 내용 분석
2. 다음 중 해당하는 것 추출:
   - 발견한 패턴
   - 주의해야 할 점
   - 해결한 문제
   - 자주 쓸 것 같은 명령

3. codify.md 파일에 추가:
   - 파일이 없으면 새로 생성
   - 있으면 적절한 섹션에 추가

4. 추가한 내용 요약해서 보여주기:

```
✅ codify.md에 추가됨:
- [추가된 내용 요약]
```

$1가 "자동"이면 AI가 판단해서 기록.
$1에 내용이 있으면 그 내용을 기록.
