## 안건 1: 외부 보안 감사 결과 공유

**박지현**: 지난달 진행한 외부 보안 감사 결과가 나왔습니다. 전체적으로는 양호한데, 개선이 필요한 사항이 몇 가지 있어요.

**김민수**: 어떤 내용들인가요?

**박지현**: 크게 3가지입니다.

첫째, 세션 관리 부분에서 세션 타임아웃이 너무 길어요. 현재 7일인데, 권고사항은 최대 24시간입니다.

둘째, API 레이트 리미팅이 없어서 DDoS 공격에 취약할 수 있어요.

셋째, 일부 로그에 민감 정보(이메일 주소)가 평문으로 저장되고 있었어요.

**김태호**: 세션 타임아웃은 고객 편의를 위해 길게 뒀던 건데, 보안이 우선이면 조정해야겠네요.

**정우성**: 레이트 리미팅은 이번 기술 부채 스프린트에서 처리하려고 했는데, 우선순위를 올려야 할 것 같아요.

### 결정 사항
- 3가지 보안 이슈 모두 Critical로 분류, 8월 내 해결
- 세션 타임아웃: 7일 → 24시간 (Remember me 옵션으로 7일 유지 가능)
- API 레이트 리미팅: 분당 100회 제한 적용
- 로그 마스킹: 이메일 등 PII 데이터 마스킹 처리

### 액션 아이템
- [ ] 김태호: 세션 타임아웃 변경 (8/16까지)
- [ ] 정우성: 레이트 리미팅 구현 (8/23까지)
- [ ] 김태호: 로그 마스킹 적용 (8/20까지)

## 안건 2: 엔터프라이즈 고객 보안 요구사항

**최영호**: 최근 엔터프라이즈 상담 중에 보안 관련 질문이 많아졌어요. SOC 2 인증 있냐, ISO 27001 있냐 이런 질문들이요.

**김민수**: 인증은 현재 없죠?

**박지현**: 네, 아직 없어요. SOC 2 Type II 인증 받으려면 최소 6개월, 비용도 5천만원 정도 들어요.

**김민수**: 비용이 만만치 않네요. 근데 엔터프라이즈 영업에 필수인가요?

**최영호**: 대기업이나 금융권은 거의 필수예요. 최근에 계약 직전까지 갔다가 SOC 2 없어서 무산된 케이스가 있었어요. ARR 1억짜리였는데...

**김민수**: 그 정도면 투자할 가치가 있겠네요. 일정은 어떻게 될까요?

**박지현**: 준비 기간 3개월, 감사 기간 3개월 해서 내년 상반기에 취득 가능할 것 같아요.

### 결정 사항
- SOC 2 Type II 인증 추진 (2025년 6월 목표)
- 9월부터 준비 시작
- 담당: CTO + 외부 컨설팅 업체

### 액션 아이템
- [ ] 박지현: SOC 2 컨설팅 업체 3곳 비교 검토 (8/23까지)
- [ ] 박지현: SOC 2 준비 로드맵 초안 (8/30까지)
- [ ] 최영호: 엔터프라이즈 고객에게 인증 추진 중임을 안내 (8/20까지)

## 안건 3: 2.0 실시간 기능 보안 점검

**박지현**: 2.0 실시간 기능에 대한 보안 점검도 별도로 했는데요, 다행히 큰 이슈는 없었어요.

**정우성**: WebSocket 연결에 JWT 인증을 적용해뒀고, 메시지 암호화도 되어 있어서요.

**김민수**: E2E 암호화는 어떻게 됐어요? 1월 기획 회의에서 논의했던 것 같은데.

**박지현**: 아, 그건 아직이에요. 엔터프라이즈 전용 기능으로 넣으려고 했는데, 개발 리소스가 부족해서 우선순위가 밀렸어요.

**김민수**: SOC 2 추진하면서 같이 개발하면 어떨까요? 보안 인증이랑 시너지가 날 것 같아요.

**정우성**: 네, Q4에 E2E 암호화 개발 계획에 넣겠습니다.

### 결정 사항
- 2.0 실시간 기능 보안 점검 완료, 이슈 없음
- E2E 암호화: Q4 개발 예정, 엔터프라이즈 전용 기능

### 액션 아이템
- [ ] 정우성: Q4 로드맵에 E2E 암호화 추가 (8/20까지)
- [ ] 박지현: E2E 암호화 아키텍처 설계 (9/15까지)

## 안건 4: 보안 교육 및 문화

**박지현**: 보안 감사 피드백 중 하나가 개발자 보안 교육이 부족하다는 거였어요. 정기적인 보안 교육을 도입하면 좋겠습니다.

**정우성**: 동의해요. 사실 보안 베스트 프랙티스를 다 알고 개발하는 건 아니거든요. 교육이 있으면 좋겠어요.

**김민수**: 외부 교육 서비스 쓸까요, 아니면 내부에서 진행할까요?

**박지현**: 기본적인 건 제가 내부 세션으로 진행하고, 심화 과정은 외부 교육으로 보내면 될 것 같아요.

### 결정 사항
- 분기 1회 내부 보안 교육 세션 도입 (9월 시작)
- 연 1회 외부 보안 교육 (OWASP 등) 지원

### 액션 아이템
- [ ] 박지현: 1차 보안 교육 자료 준비 (9/10까지)
- [ ] 정우성: 개발팀 교육 일정 조율 (8/25까지)
