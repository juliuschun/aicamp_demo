## 안건 1: 2024년 성과 리뷰

**김민수**: 연말이 다가오니 올해 성과를 한번 정리해보고 Q4 계획을 세우죠. 먼저 주요 지표부터 볼까요?

**윤재석**: 영업 지표입니다. 9월까지 ARR 12억 달성했고, 연말 목표 15억은 달성 가능할 것 같아요. 신규 고객 180개사, 엔터프라이즈 계약 5건 성사했습니다.

**송유진**: 마케팅은 웹사이트 트래픽 2배 성장, MQL 1,200건, SQL 전환율 28%입니다. 목표였던 30%는 살짝 미달이에요.

**최영호**: CS 지표는 NPS 45점으로 연초 32점 대비 크게 개선됐어요. 고객 이탈률도 월 2.5%에서 1.8%로 낮아졌습니다.

**박지현**: 제품 측면에서 2.0 성공적 런칭, 간트 차트와 실시간 협업 기능 릴리즈, 보안 강화까지 계획대로 진행됐어요.

**김민수**: 전반적으로 좋은 성과네요. 팀 모두 수고 많았습니다.

## 안건 2: Q4 OKR 확정

**김민수**: 지난번 공유한 전사 OKR 초안에 대해 논의해보죠.

**Objective 1**: 연간 매출 목표 초과 달성
- KR1: ARR 16억 달성 (현재 12억)
- KR2: 엔터프라이즈 신규 계약 3건
- KR3: Business 티어 전환율 15% 달성

**윤재석**: ARR 16억이면 남은 기간 월 1억 3천씩 성장해야 하는데, 공격적이긴 해요. 근데 연말 구매 예산 소진 시즌이라 가능성은 있어요.

**Objective 2**: 제품 안정성 및 확장성 확보
- KR1: 서비스 업타임 99.9% 유지
- KR2: E2E 암호화 기능 베타 출시
- KR3: API 응답 속도 20% 개선

**정우성**: E2E 암호화 베타는 11월 말에 가능하고, API 최적화는 12월에 진행 예정이에요.

**Objective 3**: 조직 역량 강화
- KR1: 시니어 개발자 1명 채용 완료
- KR2: 전 직원 보안 교육 100% 수료
- KR3: OKR 시스템 정착 (100% 팀 참여)

**오수현**: 채용은 2명 면접이 진행 중이고, 1명은 오퍼 단계예요. 10월 내로 합류 가능할 것 같아요.

### 결정 사항
- Q4 전사 OKR 3개 확정
- 월 1회 체크인 (매월 첫째 주 월요일)

### 액션 아이템
- [ ] 각 팀장: 전사 OKR에 연계된 팀 OKR 최종 확정 (10/11까지)
- [ ] 오수현: OKR 대시보드 Notion에 생성 (10/10까지)

## 안건 3: 2025년 사업 계획 방향

**김민수**: 내년 사업 계획을 11월까지 수립해야 해요. 큰 방향성을 논의해보죠.

**박지현**: 제품 측면에서 AI 기능 도입을 고려하고 있어요. 태스크 자동 분류, 예상 소요시간 추천 같은 거요. 경쟁사들도 하나둘 AI 기능을 넣고 있어서요.

**송유진**: 마케팅에서도 AI 어필이 많이 되더라고요. "AI-powered project management" 메시지가 요즘 트렌드예요.

**윤재석**: 영업팀에서도 AI 있냐고 물어보는 고객이 늘었어요. 없다고 하면 좀 실망하는 분위기가 있어요.

**김민수**: AI는 확실히 해야 할 것 같아요. 개발 리소스가 될까요?

**정우성**: OpenAI API 연동 정도는 Q1에 가능할 것 같고, 자체 모델은 시간이 더 필요해요.

**이서연**: 글로벌 진출도 고려해볼 만해요. 현재 해외 고객이 전체의 5% 정도인데, 영문 버전 강화하면 늘릴 수 있을 것 같아요.

**김민수**: 좋아요. 2025년 테마는 "AI 강화 + 글로벌 확장"으로 가죠.

### 결정 사항
- 2025년 사업 방향: AI 기능 도입 + 글로벌 시장 확대
- AI 기능: Q1 MVP 출시 목표
- 글로벌: 영문 버전 강화 + 해외 마케팅 시작

### 액션 아이템
- [ ] 박지현: AI 기능 로드맵 초안 (10/25까지)
- [ ] 이서연: 글로벌 진출 전략 검토 (10/25까지)

## 안건 5: 팀별 Q4 우선순위

**김민수**: 마지막으로 각 팀 Q4 핵심 우선순위 하나씩만 말씀해주세요.

**박지현 (CTO)**: SOC 2 인증 준비에 집중할게요. 내년 상반기 취득이 목표니까 연말까지 기반 작업 마무리해야 해요.

**이서연 (제품기획)**: Slack 연동 강화 기능 출시할 거예요. 고객 요청이 많았던 건데 드디어 나갑니다.

**정우성 (개발)**: E2E 암호화 개발하면서 기술 부채도 정리할 계획이에요. 연말에 깔끔하게 마무리하고 싶어요.

**송유진 (마케팅)**: 고객 사례 콘텐츠 완성하고 연말 프로모션 캠페인 진행할게요.

**윤재석 (영업)**: 엔터프라이즈 3건 클로징에 집중이요. 연말 예산 소진 시즌이라 중요한 시기예요.

**최영호 (CS)**: NPS 50점 달성이 목표예요. 고객 성공 프로그램도 시작할 거예요.

**오수현 (HR)**: 채용 마무리하고 내년 채용 계획 수립할게요. 그리고 연말 성과 평가 프로세스도 준비해야 해요.
