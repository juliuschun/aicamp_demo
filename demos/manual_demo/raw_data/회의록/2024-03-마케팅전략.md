## 안건 1: Q2 마케팅 목표 설정

**송유진**: Q1 마케팅 성과 먼저 공유드릴게요. 웹사이트 트래픽 15% 증가, 신규 가입자 820명, MQL 340건 달성했습니다. 목표 대비 MQL이 조금 부족했어요.

**김민수**: MQL이 부족했던 이유가 뭘까요?

**송유진**: 콘텐츠 마케팅 쪽은 잘 됐는데, 유료 광고 전환율이 예상보다 낮았어요. 특히 Google Ads에서 CPC가 높아져서 예산 효율이 떨어졌습니다.

**박소영**: 대신 블로그 콘텐츠를 통한 오가닉 트래픽이 40% 늘었어요. "프로젝트 관리 팁" 시리즈가 특히 반응이 좋았습니다.

**윤재석**: 영업팀에서 느끼기에는 MQL 품질도 중요한데, Q1에 들어온 리드 중에 실제 상담으로 전환된 게 25% 정도였어요. 좀 더 타겟팅된 리드가 필요합니다.

**김민수**: 그럼 Q2에는 품질 중심으로 가보죠. 양보다 질로요.

### 결정 사항
- Q2 마케팅 목표: MQL 400건 (품질 중심), SQL 전환율 30% 이상
- Google Ads 예산 축소, 콘텐츠 마케팅 강화

### 액션 아이템

## 안건 2: TaskFlow 2.0 사전 마케팅

**이서연**: 7월 2.0 출시 예정인데, 사전 마케팅을 어떻게 가져갈지 논의가 필요해요.

**송유진**: 제가 생각한 플랜은 이래요. 4월에 티저 캠페인 시작, 5월에 베타 신청 받기, 6월에 얼리버드 프로모션, 7월 런칭 이벤트 순서로요.

**김민수**: 좋네요. 경쟁사들이 최근 가격 인상을 했거든요. 우리는 기존 고객에게 2.0을 무료 업그레이드 해주면서 이 점을 어필하면 좋겠어요.

**윤재석**: "가격 동결 + 기능 대폭 업그레이드"라는 메시지가 강력할 것 같아요. 기존 고객 리텐션에도 도움 되고요.

**박소영**: 사례 기반 콘텐츠도 준비하면 좋을 것 같아요. 베타 테스트 고객사 중에서 성공 사례를 먼저 만들어두면요.

**이서연**: 베타 테스트 5월에 예정되어 있으니까, 6월에 사례 콘텐츠 제작 가능할 것 같아요.

### 결정 사항
- 2.0 사전 마케팅 일정 확정 (4월 티저 → 5월 베타 → 6월 얼리버드 → 7월 런칭)
- 핵심 메시지: "가격 동결, 실시간 협업으로 완전히 새로워진 TaskFlow"
- 베타 테스트 고객 사례 콘텐츠 6월 제작

### 액션 아이템
- [ ] 송유진: 2.0 런칭 마케팅 상세 플랜 (3/25까지)
- [ ] 박소영: 티저 콘텐츠 초안 작성 (4/1까지)
- [ ] 이서연: 베타 테스트 고객사 리스트 마케팅팀 공유 (4/8까지)
