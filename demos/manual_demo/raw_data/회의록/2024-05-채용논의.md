**장소**: 회의실 A

## 안건 1: 채용 현황 및 필요성

**박지현**: 2.0 출시를 앞두고 개발팀 리소스가 많이 부족합니다. 현재 개발자 7명인데, 최소 2명 더 필요해요. 특히 시니어 백엔드 개발자가 급해요.

**정우성**: 실시간 기능 개발하면서 느낀 건데, 분산 시스템 경험 있는 분이 꼭 필요해요. 저희 팀에 그 분야 전문가가 없어서 제가 공부하면서 개발하고 있거든요.

**김민수**: 시리즈 A 투자금이 들어온 상황이니까 채용 예산은 있어요. 다만 좋은 인재를 어떻게 뽑느냐가 문제죠.

**오수현**: 현재 채용 공고가 올라간 지 3주 됐는데, 지원자가 15명 정도 들어왔어요. 근데 시니어급은 3명뿐이고, 그중 2명은 연봉 기대치가 저희 예산을 많이 넘어요.

**박지현**: 연봉 기대치가 얼마나 됐어요?

**오수현**: 1억 2천 정도요. 저희가 제시할 수 있는 최대가 9천이라고 했더니 거절하셨어요.

**김민수**: 스톡옵션으로 보완하면 어떨까요? 현금 연봉은 조금 낮더라도 성장 가능성을 어필하면요.

### 결정 사항
- 시니어 백엔드 개발자 1명 + 미드레벨 풀스택 1명 채용 목표
- 연봉 상한: 9천만원 + 스톡옵션 0.3% 패키지 제안 가능
- 채용 채널 다변화 필요

### 액션 아이템
- [ ] 오수현: 스톡옵션 포함 보상 패키지 설계 (5/24까지)
- [ ] 오수현: 채용 채널 확대 (원티드, 로켓펀치 외 링크드인 추가) (5/22까지)

## 안건 2: 채용 프로세스 개선

**박지현**: 지금 채용 프로세스가 너무 오래 걸려요. 서류 → 기술면접 → 과제 → 컬처핏 → 최종면접까지 3주 넘게 걸리더라고요.

**오수현**: 좋은 후보들이 다른 회사 오퍼를 먼저 받아서 이탈하는 경우가 있었어요. 프로세스 단축이 필요할 것 같아요.

**정우성**: 과제 전형이 시간이 많이 걸리는데, 라이브 코딩으로 대체하면 어떨까요? 2시간 내로 끝낼 수 있으니까요.

**박지현**: 동의해요. 과제 전형은 후보자 입장에서도 부담이 크거든요. 라이브 코딩으로 바꾸고, 기술면접과 같은 날 진행하면 전체 일정을 1주일로 줄일 수 있을 것 같아요.

**김민수**: 좋아요. 컬처핏도 기술면접 후에 바로 진행하면 되겠네요. 제가 컬처핏은 시간 맞춰서 들어갈게요.

### 결정 사항
- 채용 프로세스 단축: 3주 → 1주
- 과제 전형 폐지, 라이브 코딩으로 대체
- 기술면접 + 라이브 코딩 + 컬처핏 원데이로 진행

### 액션 아이템
- [ ] 정우성: 라이브 코딩 문제 세트 준비 (5/27까지)
- [ ] 오수현: 채용 프로세스 가이드 문서 업데이트 (5/24까지)
- [ ] 박지현: 기술면접 질문 리스트 정리 (5/27까지)

## 안건 3: JD(Job Description) 검토

**오수현**: 현재 JD가 너무 요구사항 위주라 매력이 없다는 피드백이 있었어요. 회사의 기술 스택이나 개발 문화를 더 어필하면 좋겠어요.

**박지현**: 저희 기술 블로그를 시작하는 것도 방법일 것 같아요. 실시간 협업 기능 개발기를 시리즈로 쓰면 기술력 어필에도 되고, 채용에도 도움이 될 거예요.

**정우성**: 좋은 생각이에요. 개발자들이 글 쓸 시간이 문제긴 한데, 2.0 출시 후에 하나씩 써볼게요.

**김민수**: 개발 문화 측면에서는 코드 리뷰 문화, 스프린트 방식, 기술 의사결정 과정 같은 것들을 JD에 넣으면 좋겠어요. 요즘 개발자들이 그런 거 많이 보더라고요.

**오수현**: 네, 반영해서 JD 새로 작성해볼게요. 개발팀에서 내용 검토해주세요.

### 결정 사항
- JD 리뉴얼: 기술 스택 + 개발 문화 상세 기술
- 기술 블로그 8월 런칭 목표

### 액션 아이템
- [ ] 오수현: JD 초안 리뉴얼 (5/27까지)
- [ ] 정우성: JD 기술 부분 검토 (5/29까지)
- [ ] 박지현: 기술 블로그 플랫폼 선정 (6/30까지)
