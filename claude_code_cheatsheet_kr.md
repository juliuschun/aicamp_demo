## 1. 필수 단축키

| 단축키        | 기능            | 쉬운 설명          |
| ------------- | --------------- | ------------------ |
| `Enter`       | 전송            | 카톡 보내기        |
| `Esc`         | 중단            | 멈춰!              |
| `Esc` + `Esc` | 되돌리기        | Ctrl+Z 강화판      |
| `Ctrl+C`      | 강제 종료       | 전원 끄기          |
| `Ctrl+R`      | 전체 보기       | 스크롤 없이 전체   |
| `Ctrl+V`      | 이미지 붙여넣기 | 스크린샷 공유      |
| `Shift+Tab`   | 자동 승인       | "알아서 해줘" 모드 |
| `\` + Enter   | 줄바꿈          | 계속 쓰기          |

## 4. 프롬프트 공식

### 좋은 프롬프트 = 5요소

```
1. 목표:     무엇을 만들 것인가?
2. 구체성:   어떤 모습이어야 하는가?
3. 제약조건: 어떤 기술/스타일을 쓸 것인가?
4. 성공기준: 어떻게 되면 성공인가?
5. 제외사항: 무엇을 하지 말아야 하는가?
```

### 템플릿

```
[목표] ~를 만들어줘.

포함할 내용:
- 항목1
- 항목2

제약조건:
- 기술: React/Python/HTML
- 스타일: 파란색, 모던

성공기준:
- 버튼 클릭 시 ~가 되어야 함

제외:
- ~는 하지 마
```

## 5. 바이브 루프 (6단계)

```
1. 목표 설정    "무엇을?"
      ↓
2. 범위 정하기  "어떤 파일?"
      ↓
3. AI 생성     [자동]
      ↓
4. 바이브 체크  "실행해보기"
      ↓
5. 검증        "에러 없나?"
      ↓
6. 저장        "커밋!"

   🔁 2-5 반복
```

## 6. 문제 해결 빠른 가이드

| 증상              | 원인                 | 해결                                       |
| ----------------- | -------------------- | ------------------------------------------ |
| command not found | 미설치               | `npm install -g @anthropic-ai/claude-code` |
| 응답 느림         | 네트워크/복잡한 요청 | 요청 나누기, 인터넷 확인                   |
| 코드 안 됨        | 에러                 | 에러 메시지 클로드에게 전달                |
| 같은 실수 반복    | 컨텍스트 혼란        | `/rewind` 후 새로 시작                     |

## 8. 자주 쓰는 프롬프트 예시

### 웹페이지 만들기
```
간단한 소개 페이지를 만들어줘.
- 상단: 회사 로고, 메뉴
- 중앙: 서비스 설명 3가지
- 하단: 연락처, 주소
- 색상: 파란색 테마
- HTML/CSS만 사용
```

### 데이터 분석
```
sales.csv 파일을 분석해줘.
- 월별 매출 그래프
- 상위 5개 제품
- 전월 대비 증감률
- Python + matplotlib 사용
```

### 자동화 스크립트
```
매일 아침 9시에 실행되는 스크립트:
- 어제 주문 데이터 수집
- 요약 리포트 생성
- Slack에 자동 전송
```

### 버그 수정
```
@src/login.js 이 파일에서
로그인 버튼 클릭해도 반응이 없어.
콘솔에 이런 에러가 나와:
[에러 메시지 붙여넣기]
```
